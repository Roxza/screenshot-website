## 📝 Running the App
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
